# Selector Copy

指定した XPath にマッチする要素のテキストをワンクリックでクリップボードへコピーする Chrome 拡張機能。
「コピー設定 (preset)」を複数管理でき、各 preset は複数の XPath を対象にできる。

## 仕様

- ツールバーアイコンのポップアップに preset 一覧を表示。ボタンクリックでコピー実行
- 各 XPath が複数要素にマッチした場合は **全件** 取得
- 集めたテキストは preset ごとに設定した **区切り文字** で連結してコピー（デフォルトは改行 `\n`。`\n`=改行 / `\t`=タブ を解釈。`, ` や ` | ` なども指定可）
- **テンプレート**（任意）: `{1}` `{2}`… に各 XPath の結果（区切り文字で連結したもの）を差し込んで出力できる。例: 区切り文字 `,`・テンプレート `価格: {1}` で XPath1 が `100`,`200` にマッチ → `価格: 100,200`。テンプレートが空なら全結果を区切り文字で連結（従来動作）
- preset はオプションページで追加 / 編集 / 削除（`chrome.storage.sync` に保存）
- **ページから XPath を取得**: ポップアップのボタンから要素ピッカーを起動。ホバーで要素をハイライトし、クリックするとその要素の XPath を生成してクリップボードへコピー（`Esc` でキャンセル）。一意な `id` があれば `//*[@id="..."]`、なければ位置インデックス付きの絶対パスを生成

## インストール（開発版の読み込み）

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」→ このディレクトリを選択

## 使い方

1. ツールバーの拡張機能アイコンを右クリック →「オプション」、またはポップアップの ⚙︎ から設定を開く
2. preset を追加し、名前と XPath（複数可）を入力（自動保存）
3. コピーしたいページでアイコンをクリック → preset ボタンを押すとコピー

## 構成

| ファイル | 役割 |
| --- | --- |
| `manifest.json` | MV3 定義（権限: storage / scripting / activeTab） |
| `popup.html` / `popup.js` / `popup.css` | preset 一覧の表示とコピー実行 |
| `options.html` / `options.js` / `options.css` | preset の管理 UI |
| `lib/storage.js` | `chrome.storage` への preset 読み書き |
| `lib/extract.js` | 対象タブに注入して XPath を評価しテキストを返す |
| `lib/picker.js` | 対象タブに注入する要素ピッカー（ホバーハイライト + クリックで XPath 生成・コピー） |
| `icons/` | アイコン（暫定プレースホルダ） |
